Pensamiento Matematico

Este proyecto esta enlazado con la UAC de Humanidades
dicho proyecto es una tabla comparativa de instituciones las cuales se encargan
de ayudar a personas con problemas de adiccion graves estas instituciones
los tratan de la mejor forma posible buscando evitar que la persona sufra un daño mayor
y tratando de tener siempre el mejor resutado.

Cultura Digital

Durante este tercer parcial estudiamos el lenguaje de programación HTML y CSS los cuales
fueron utilizados para la realización de esta pagina web dichos codigos funcionan con
etiquetas como "< p >< /p >" para poner un parrafo o "< ul >< /ul >" o "< ol >< /ol >"
para crear listas ordenadas o desordenadas.

Ciencias Sociales

Para esta proyecto se realizaron una serie de preguntas que fueron proporcionadas por la
profesora, estas cada integrante estuvo acargo de una pregunta las cuales fueron contestadas
basandose en lo que se vive en nuestras comunidades y en nuestro pais con respecto a las drogas.

Taller de Ciencias

Para este proyecto realizamos un prototipo didactico el cual funcionaba con energia cinetica
dicho prototivo funcionaba con la ayuda de un motor dínamo el cual genera la suficiente energía
como para encender 14 leds, este proyecto busca la reduccion de gases de efecto invernadero
así como el uso de combustibles fociles y otros que puedan dañar la capa de ozono la cual nos protege
de una enorme cantidad de rayos UV emitidos por el sol los cuales son dañinos para la piel.

Lengua y Comunicación

En este proyecto realizamos una reseña la cual trataba de como se realizaron los demas proyectos
utilizamos el libro de texto como apoyo.

Íngles

Para este proyecto realizamos un fragmento de comic el cual fue creado a vase de los verbos going to
que se utiliza para planear algo que haras en el futuro, por lo cual hicimos una lista de fraces de
cosas que nos gustaria hacer en un futuro.

Conservación de la Energía

En este proyecto realizamos una investigación sobre el funcionamiento del cerebro para ello analizamos
como funcionaba un cerebro bajo la influencia de las drogas y como funcionaba de manera normal y pudimos
notar el cambio drastico que este sufre.

Humanidades

En este proyecto realizamos un decalogo para dicho decalogo buscamos problemas en nuestras comunidades
con respecto a las drogas y que podemos hacer para evitr ese tipo de problemas los cuales generan un enorme
daño.
